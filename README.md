# CuZr-MD-DMS

Project for Paper 2 calculations: molecular-dynamics dynamic mechanical spectroscopy
(MD-DMS) simulations of nearly constant loss and fast relaxation in Cu-Zr metallic glass.

## Strategy

- Python workflow without pyiron
- LAMMPS input files generated by Python
- MACE_C as the main production candidate
- EAM_2019 as a classical comparison model
- Docker base image without prebuilt LAMMPS
- LAMMPS built on the cloud GPU machine with ML-IAP and Kokkos CUDA

## Current status

This repository is initially a minimal project skeleton. Scripts, LAMMPS templates,
and analysis tools will be added step by step.
